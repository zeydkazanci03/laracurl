# Changelog

All notable changes to `laracurl` will be documented in this file.

## 1.0.0 - 2025-10-26

### Added
- İlk sürüm
- Tüm HTTP metodları desteği (GET, POST, PUT, PATCH, DELETE, HEAD, OPTIONS)
- JSON, Form Data, Multipart desteği
- Dosya yükleme özelliği
- Bearer Token authentication
- Basic ve Digest authentication
- Proxy desteği
- Cookie yönetimi
- SSL doğrulama kontrolü
- Header yönetimi
- Timeout ayarları
- Yönlendirme kontrolü
- Response helper metodları
- Laravel Facade desteği
- Fluent API interface
- Detaylı dokümantasyon
